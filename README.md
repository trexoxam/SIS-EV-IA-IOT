# SISTEMA INTELIGENTE DE EVALUACIÓN Y SUPERVISIÓN

## DESCRIPCIÓN

El proyecto denominado "Sistema Inteligente de Evaluación y Supervisión" tiene como finalidad automatizar el proceso de registro, programación y supervisión de aspirantes que realizarán un examen.

El sistema permitirá registrar usuarios, recopilar información mediante cuestionarios previos, asignar citas de manera inteligente utilizando algoritmos de Inteligencia Artificial y supervisar el desarrollo de los exámenes mediante cámaras conectadas a dispositivos IoT.

Durante la aplicación del examen, la Inteligencia Artificial analizará movimientos y comportamientos del aspirante para detectar patrones relacionados con posibles conductas sospechosas, distracciones o irregularidades, generando reportes para los administradores del sistema.

---

## ARQUITECTURA

El proyecto se desarrollará utilizando una arquitectura Cliente-Servidor de tres capas:

### Capa de Presentación
Interfaz web para aspirantes y administradores.

### Capa de Negocio
Servicios encargados de la lógica del sistema, gestión de citas y procesamiento de datos.

### Capa de Inteligencia Artificial
Módulos encargados del análisis conductual y detección de movimientos mediante visión por computadora.

### Capa de Datos
Almacenamiento de usuarios, resultados, citas y reportes.
